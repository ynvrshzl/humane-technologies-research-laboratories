# Cyberethos
_**Ethics + Cybernetics.** A humane ethical technology research project_

![](https://images.pexels.com/photos/724994/pexels-photo-724994.jpeg)

## Proposition - An emerging field of study is required for the increasing interconectedness of technology and human beings.
We envision a collaborative, open-source, decentralized, human effort. No single individual can solve the problems in our world, but collectively, we can work towards reduction of suffering, utilizing the power of computation, as a tool to find answers, not technology as the answer.


## "How do we create more ethical, humane technologies that help us solve real-world problems?"

This is the question that drives this entire field. we're not sure if this is a field of study, or a research group, or a name of a future open source software development team... but this is a pretty badass starting point!

This repository contains documentation, for a group of planned software, for technologies which are attuned to the somatic human response, interface between human and a machine. Humane, ethical technologies.

## Aliases for this project
- **"Cyberethos"** - Capturing the "ethos" of the technology.
- **Humane acoustic somatic research technology laboratories** - Since this will someday be a decentralized, open-source, collective of software architecture.
- **"Homotechno Mechanics"** - The functional relationship at which the human-machine interconectedness is used for healing.
