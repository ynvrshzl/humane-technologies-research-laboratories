# Humane Ethical Technology Research Project 

![](https://images.pexels.com/photos/724994/pexels-photo-724994.jpeg)

> "How do we create more ethical, humane technologies that help us solve real-world problems?"

This is the question that drives this entire field. we're not sure if this is a field of study, or a research group, or a name of a future open source software development team... but this is a pretty badass starting point!

This repository contains documentation, for a group of planned software, for technologies which are attuned to the somatic human response, interface between human and a machine. Humane, ethical technologies.
